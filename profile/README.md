# Bright Spark Foundation Community Hub 🌟

Welcome to the central repository for the **Bright Spark Foundation** community! We are a student-led initiative from TJHSST dedicated to introducing middle and elementary school students to electronics, hardware engineering, and coding.

This repository serves as our team's main hub for collaboration, curriculum development, and resource sharing.

🔗 **Live Website:** [bright-spark-foundation.vercel.app](https://bright-spark-foundation.vercel.app)

## 📁 What's Inside

* **`/curriculum`** *(Coming Soon)*: Lesson plans, wiring diagrams, and code snippets for Arduino, Raspberry Pi, and micro:bit projects.
* **`/assets`**: Official branding assets, logos, and flyer templates.
* **`/planning`**: Meeting notes, outreach templates, and operational roadmaps.

## 🚀 Our Mission

We believe hands-on engineering should be accessible to every young learner. By designing intuitive, project-based hardware bootcamps, we aim to spark a lifelong passion for STEM in our local community.

## 🤝 Getting Involved

If you are a member of the team or a volunteer mentor:
1. Check the **Issues** tab for open tasks or documentation needs.
2. Create a new branch for any curriculum updates you are drafting.
3. Submit a Pull Request for review before merging major guides.
